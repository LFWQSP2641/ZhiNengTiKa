## 开发

### 编译

#### 编译环境
>Windows: Qt 6.5.2 + MSVC2019

>Android: Qt 6.5.2 + NDK 25.1.8937393 + JDK 17

#### 如何编译

```bash
cd ZhiNengTiKa
git submodule update --init --recursive
```

使用 Qt Creator 打开 `ZhiNengTiKa.pro` 或 `ZhiNengTiKaQML.pro`
